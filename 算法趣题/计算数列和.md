## for-loop 迭代
```python
def sum(list):
    sum = 0
    for i in range(len(list)):
        sum += list[i] 
    return sum
```

## 线性递归 linear recursion
```python
def sum_1(list):
    if len(list) == 0:
        return 0
    else:
        return list[0] + sum_1(list[1:])
```
### 线性进阶:计算这个序列的前n项和
```python
def sum_2(n,list):
    if n == 0:
        return 0
    else:
        return list[n-1] + sum_2(n-1,list)
```
