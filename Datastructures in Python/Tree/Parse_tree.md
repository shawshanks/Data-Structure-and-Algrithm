## 名称：分析树
###目标： 使用分析树 来进行数学表达式计算
###方法（步骤）
    1. 建立分析树
    2. 计算树中的表达式
    3. 从树中恢复原来的数学表达式

## 1.建立分析树
### 4类不同的符号 token
- 左括号

左括号是一个新的表达式的开始。我们为对应的表达式 建立一颗 新树。

-  右括号
  
右括号是一个表达式的结束。

-  运算数 operand
  
运算数是叶节点，是 operator 运算符 的子节点

-  运算符 operator

运算符 将会有 左右子节点
